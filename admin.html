⚙️ GitHub 连接配置

🔗 连接信息

GitHub → Settings → Developer settings → Personal access tokens → 勾选 repo

📖 使用说明

1. 在 GitHub 创建仓库，开启 GitHub Pages

2. 生成 Token（只需 repo 权限）

3. 填入信息 → 保存 → 测试连接

4. 看到绿色成功提示后开始管理内容

5. 修改后点击「发布所有更改」→ 约1分钟生效
